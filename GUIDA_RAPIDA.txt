## Installazione

1. Installa le dipendenze Python:
   pip install click pydantic requests

2. Installa il pacchetto:
   pip install -e .

## Utilizzo Base

### Test in modalita sandbox (sicuro):
python -m xwirless scan --sandbox tests/samples/sample_iwlist.txt --format json

### Test in modalita dry-run (simulazione):
python -m xwirless scan --dry-run --format md

### Scansione reale (SOLO su reti autorizzate):
python -m xwirless scan --format all --save-db

### Mostra informazioni:
python -m xwirless info
python -m xwirless version

### Gestione inventario:
python -m xwirless inventory
python -m xwirless diff scan_id1 scan_id2

## Formati Output

- JSON: Dati strutturati per analisi

## Esempi di Report

I report includono:
- Metadati (timestamp, interfaccia, versione)
- Informazioni rete (SSID, BSSID, canale, segnale)
- Tipo di crittografia
- Statistiche di riepilogo
